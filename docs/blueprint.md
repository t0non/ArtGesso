# **App Name**: Art Gesso Landing Page

## Core Features:

- Estrutura da Landing Page: Estrutura HTML5 semântica com cabeçalho, navegação, seções principais (serviços, gatilhos de confiança, locais) e rodapé, otimizada para SEO.
- Vitrine de Serviços: Exiba os serviços oferecidos (Rebaixamento de Teto, Sancas de Gesso, etc.) usando um layout de grade ou cartão com ícones relevantes.
- Exibição de Contato e Localização: Mostre dois locais de atendimento com endereços (Rua Humaitá, Rua Progresso), números de telefone e links diretos para WhatsApp e Google Maps.
- Botões de Chamada para Ação (CTA): Botões proeminentes como 'Solicitar Orçamento' e 'Fale com um Especialista no WhatsApp' com links diretos para informações de contato.
- Otimização de SEO: Implemente dados estruturados (Schema.org) do tipo LocalBusiness, incluindo endereço e informações de contato.
- Chat com Art Gesso: A ferramenta de IA analisará o conteúdo da página inicial e fornecerá sugestões sobre qual número de telefone ou informação da página inicial pode melhor responder às perguntas do usuário. Ferramenta de IA

## Style Guidelines:

- Cor primária: Cinza claro (#E0E0E0) para representar a estética limpa e profissional do gesso.
- Cor de fundo: Branco sujo (#FAFAFA) para criar uma tela limpa e convidativa.
- Cor de destaque: Verde (#4CAF50) para botões de chamada para ação, destacando a opção de contato do WhatsApp.
- Fonte do corpo e título: 'PT Sans', um sans-serif humanista que oferece uma aparência moderna com acessibilidade e calor, adequado para títulos e texto do corpo.
- Design responsivo Mobile-first, garantindo uma experiência de visualização ideal em vários dispositivos (desktops, tablets, smartphones).
- Use ícones claros e relevantes (FontAwesome ou SVG inline) para representar visualmente os serviços oferecidos em um formato amigável.
- Efeitos sutis de foco nos botões e cartões de serviço para melhorar a interação do usuário.
